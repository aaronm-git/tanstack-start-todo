# Authentication (Clerk)

This app uses Clerk for authentication.

## Configuration

- Local env: set `VITE_CLERK_PUBLISHABLE_KEY` in `.env.local`.

## Where to look in the code

- Search for Clerk integration and providers in `src/`.
- Search for the publishable key usage: `VITE_CLERK_PUBLISHABLE_KEY`.
